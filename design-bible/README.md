## The Standard

The v2 landing page (`/v2`) is the reference implementation. Every new page must match or exceed this level of:
- Visual sophistication (cinematic overlays, parallax depth, BTS photography)
- Interaction quality (video-on-hover, cinematic player modals, smooth scroll)
- Conversion architecture (dual-path CTAs, lead magnet, scarcity signals)
- Performance discipline (IntersectionObserver gating, RAF throttling, no infinite waste)
